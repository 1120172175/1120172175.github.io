开学第一周生存报告

发表于 2019-03-02

date: 2019-03-02 20:44:10

为什么要写这个呢，因为今天的精神状态比预想的要好很多。早上还是按部就班的起床，记单词，然后去干活，十点半匆匆去吃了午饭，然后就背着重重的包去赶地铁了，下午完全不困，兴致勃勃而且精神地差不多听完了全程，然后傍晚挤上了地铁，不饿不累也不困。

那正常情况下是什么样呢，大抵是经过了一个没有午睡的中午后，下午会有点困（或者很困），然后傍晚又困又饿地靠着地铁俯扶杆全程睡觉回学校吧。

这一周，早上闹钟响后都不会再很困，而是机灵地起来想着这一天要干啥（有一天多睡了，但也不是因为困），然后去北湖听着歌小碎步跑一圈醒神。中午小睡一会，下午也不会再像以前那样困了，有时候甚至会十分精神，晚上继续该干活干活。

我想，这也并不是突然打了鸡血吧。我将其归结于开学第一周良好的作息生态的维持。这一周基本都保持着十一点睡觉六点起床，为什么用保持而不用坚持呢，因为十一点就困了，六点闹钟响了也不困，就起来了呗（其实还有就是六点不起来单词就背不完了…）。可能就会想，六点起来为什么会不困呢，以及，六点起来早上不会很困吗？想到这个的时候，往往会忽略十一点睡这个东西。

记得开学前两天，哦不，其实寒假想这样想了很久了，刚回学校的那两天，十一点爬上床，往往要十二点以后甚至一点钟才睡得着，第二天甚至起的更晚了，记得那两天都是十点多以后才起的床，有两天直接睡满了12个小时，醒来头脑还是有点昏昏沉沉的，那是一种挫败感啊，真的很想早点起来但是还是睡趴了（都是寒假睡的…），但是，一直坚持十一点左右爬上床，直到有一天，闹钟一响手动扇了一巴掌，总算是见到了天还没亮的早晨。然后，然后八点就倒下了…睡到了九点继续干活。

又过了两天，晚上十一点自然就想睡觉了，早上也慢慢的不困了，下午也越来越有精神了，一切在慢慢摆脱刻意而变得越来越自然。

我越来越觉得只有自然而然的事情可以在不带给自己很多压力的同时又能做很久，所谓坚持，有时候是压迫，精神上的被剥削，我觉得高考算是一个；而有时候就是在别人看来是坚持而在自己看来不过是一件很自然的事情罢了，比如乐器，我这种单纯想学一门乐器的人肯定不能和喜欢音乐的人比，我也确实没有学下去，最开始的坚持也不能让它在我的生活中变得自然，遂放弃。还有读书，我现在也不读经典，就喜欢读些有趣的书，比如《生命册》，读了两遍还想读第三遍，因为觉得它有趣，骆驼也很酷，比如最近看的江南三部曲，读了后两部觉得很有意思，谭端午谭功达父子俩也经常让我若有所思，但是豆瓣知乎的书友并没有给出多好的评价。我不懂表现艺术，我也不懂故事的叙述什么的，相反，对《围城》的评价就太高了，我也确实觉得围城写得好，很有韵味更有内涵，钱钟书先生的话术也确实精巧，但是我更喜欢读前面的那两本书，因为至少目前位置，对我而言，读书里的趣味更吸引着我（嘿嘿此处不接受反驳），这样才能让我感觉到很自然，很自在，很快乐。对比之前多次失败的早起计划，想过来还是因为它既没有能够在精神上压迫我（因为我起不来其实完全没有后果），而且也很不自然（因为那样起来以后真的很困啊，晚睡早起简直折寿），所以以前的时候，早起对我而言是一件非常需要坚持的事情，而且没有必要，所以到底是在满足哪门子虚荣心呢？但是当一件事在自己的生活中被变得很自然的时候，就像吃饭一样，会是一件很爽的事情，很多其他的事情也都会进入一个积极的循环，感受自知。或许，还有更多的事情值得去这样尝试吧，反正也没啥代价。

写到此处，这个flag已经给我一种摇摇欲坠的感觉，反正，能这样保持，多一天就赚一天，啥时候又会起不来床已经不重要了

第五周小结
