反思

发表于 2018-12-23

date: 2018-12-23 08:18:43

昨天考完试之后还是比较开心的，算是暂时讨好一下自己吧。但是现在要开始正式范式昨天离散考试的失败了。

首先就是考试的时候为什么那些题没有做出来，原因是什么？现在想想自己复习的思路以及方法都错了，没有去成体系地复习，而且存在侥幸心理，复习的时候不认真效率不高，做的题不够。

  1. 复习的整体思路。这一次离散的复习就是单纯地看PPT记概念，直到最后两天才开始做题，时间明显不足，总体的复习进度没有规划好，导致最后比较手忙脚乱，最后两天效率不高。以后一定要在复习开始之前将总体的计划详细一点列出来，看完PPT之后分方向去复习，比如专门复习某一章节的证明题，或者计算题。

  2. 存在侥幸心理。看到去年的题主要没怎么考证明题，所以证明题的练习就不够，导致最后，今年的考试全是证明题，做起来很吃力，最后也没有考好，问题所在。以后一定一定一定不能在没有根据的情况下盲目判断，就比如今年之后一套往年的卷子，就不能根据这套卷子去确定复习的重心，侥幸心理害死人。以后的复习一定要系统，不能乱搞。

  3. 复习的效率不高。这一次离散的复习，很多时候都不认真，虽然再看PPT，但是脑子并没有在思考问题，只是机械地在看而已，并没有太把思维性放在心上，导致最后一道稍微转点弯的大题都想不出来，不知道往哪个方向去想，这就是思考的少的下场，以后要杜绝这样的复习状态的再次出现。

  4. 做题不够。这次离散考试很多题都是教材上的原题，自己没有花功夫去做题，不然那几道做的不好的题就都不是问题了，就是在于考试之前没有下狠功夫去复习，准备不充分。

最后，离散没考好肯定不是偶然的，是一个学习上的问题的具体体现，有关学习的心态，学习的方法，以及学习的毅力。这三样在最近这一个月都慢慢地不是很上心了，说白了就是变懒了。不见棺材不落泪也太真实了，接下来一定要好好学习！

下午去听了UCLA学姐暑研的分享，感觉还是不错的。

今日早起后的一点点小感悟

发表于 2018-12-14

date: 2018-12-14 09:39:41

今天六点多就起来了，现在是九点半，困意开始弥漫。
为什么这个点要写东西呢，因为昨天睡得晚，今天又起得早，精神实在有一点点恍惚，想歇一下。今天开始看杨鹏长难句，确实感觉很多东西方法的重要性，里面讲到的很多东西真的是戳中要害了，包括阅读的时候老是忍不住回看，心里默念句子以翻译的形式去理解句子。现在看来，这都是很错误的东西呀，被框在了应试教育的笼子里，追求的是分数，学的东西都是人native speaker不会用的东西，厉害的中国人编外语教材，到头来仍然是蹩脚的，现在回看学了这么多年的英语，实用性甚少，学了这么多年的阅读，我学会了如何去阅读吗？没有，我学到是去做阅读题，如何拿分。我想起了教材上教的没有一个外国人会用的just so so，老外都说It’s ok It’s fine，那教我们just so so用来对话，见识浅薄的我实在看不到意义何在？同样的还有Mr.k告诉我们的，没有美国人会用one by one，他们只会说one at a time。我又想到了俄罗斯高考的汉语考试，一看那种汉语就是俄罗斯人写的，而不是在中国长大的人写的，这是不是和外国人看中国人的高考英语感觉有点像呢？现在想想，学了这么多年外语，我学到的究竟有多少东西是陈年旧物，究竟有多少东西本身就是误会呢？究竟是什么出了问题呢，谈教育体制太大了，包括我在内的千千万万人都在吐槽之，设计师们又怎么不会意识不到问题的存在呢，我相信这个国度有着千千万万的有识之士在努力改变一些东西，让教育更好，同时我也猜测，有千千万万保守者是投鼠忌器的，他们或许曾经也是那些有识之士的一员，谁知道时间带来的是什么呢，作为一名安分守己的大学生，我也不知道我能为教育变得更好做些什么，只能感受其中慢慢的变化，只能在不满时默默吐槽，我感受不到有识之士们在大环境下的无奈和艰难险阻，或许我现在可以在心里为你们加油吧！

我永远都忘不了高考结束后的两天我来到了北京考自主招生的经历，那个培训班，那个北京四中的名师给我留下的印象之深，真的难以用言语形容，我第一次感觉到原来教育资源的差距可以那么那么大，回酒店班主任问我今天课上的怎么样？我回答“特别特别好，光这一节语文课，这钱花的我都觉得值了”。这边的语文可以这么这么棒，他教的不仅仅是超赞的考试技巧，还有诗情画意，还有生活中无处不在的美，还有发人深省的辩证法，还有人生，还有哲理。到后来，我认识了高中人大附的同学，十一学校的同学，听着他们对自己母校的描述，真的就在心里想：我以后一定要让我的孩子在北京接受优质的教育，让他在最自由最好的环境中成长成为他自己想成为的人，而不是像我一样，需要我自己去拼尽全力创造很多人生下来就拥有的东西。很多事情本就是不公平的，很多时候却又是公平的，何来生活的惬意，自由的思想与灵魂？不是捡来的，是努力出来的。大学不长，就几年，但如果这几年通过自己努力就能拥有好的生活习惯，坚持学习看书自律的态度，习惯对生活充满激情，对待别人充满热情和真诚，知道自己喜欢干什么，找到自己挚爱的人并懂得珍惜懂得爱，那多好啊，想想都刺激。五年后和老友聊起自己的大学生活，故事三天三夜都说不完，那些最真挚，最热血，最拼的日子，绝对不是在寝室坐出来睡出来的呀！你要加油呀！！！

码的字渐渐有了浓浓的鸡汤味，但是要知道的是，这些字不是写给你看的，是刘勇写给他自己和与他并肩奋斗的人看的。
